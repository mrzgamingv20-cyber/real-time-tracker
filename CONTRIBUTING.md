# Contributing to Real Time Tracker

## Development Setup

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Commit changes: `git commit -am 'Add new feature'`
4. Push to branch: `git push origin feature/your-feature`
5. Submit a pull request

## Code Style

- Follow Kotlin conventions
- Use meaningful variable names
- Add comments for complex logic
- Format code with IDE's built-in formatter

## Testing

- Test on physical device when possible
- Test with different Android versions
- Verify permissions work correctly

## Commit Messages

Use descriptive commit messages:
- `feat: add object tracking feature`
- `fix: resolve camera preview issue`
- `refactor: improve performance`
- `docs: update README`

## Questions?

Open an issue or discussion for questions!
